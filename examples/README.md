# metaLncRNA Examples

This directory contains resources to help you get started with metaLncRNA.

## Contents
- `sample.fasta`: A small dataset for testing the pipeline.
- `run_example.sh`: A shell script to run the full ensemble pipeline.
- `config_template.yaml`: A template for custom configurations (weights, tool paths).

## Quick Start
To run the analysis:
```bash
chmod +x run_example.sh
./run_example.sh
```

Ensure you have run `metalncrna setup` before executing the example script.
