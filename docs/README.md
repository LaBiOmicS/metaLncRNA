# metaLncRNA Documentation

Welcome to the official documentation for **metaLncRNA**, a modern, modular, and extensible meta-predictor for identifying Long Non-coding RNAs (lncRNAs).

## Quick Overview

`metaLncRNA` resolves the "reproducibility gap" in lncRNA identification by automating environment management and providing a robust consensus prediction through weighted soft-voting of 7 diverse computational tools.
